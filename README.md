# MSC Tools for Fusion 360 — Interactive Mock

The mock application only — no landing page. All data simulated.

## Structure (must stay exactly like this)

```
index.html
css/style.css
js/app.js
```

## Deploy to GitHub Pages

Push all files and both folders to the repo root on `main`
(Pages settings stay as-is: deploy from `main` / root).
Open the site root URL — it must NOT end in `.js` or `.css`.

After pushing, hard-refresh: Ctrl/Cmd+Shift+R (Pages caches).
